What is a Shell ?
A shell is a programm that acts as an interface between the user and the Linux operating system (kernal)

There are different varieties of shell based on the operating system you use

-- bydefault shell is called as sh (shell)
-- the one which comes is linux is called as bash (bourne shell)
-- in mac os it is called as zsh
-- there is another variant also called as csh (corn shell)
