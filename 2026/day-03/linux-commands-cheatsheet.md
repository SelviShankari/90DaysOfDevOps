1.) ps :- This command lists out all the processes. It is like a photo that is clocked. I shows the processes.

2.)top :- This comamnd lists out all the running process but in cctv mode. Like i shows the cpu usage , memory usage in real-time.

3.) kill :- kill command is used to kill any running process.

4.)nohup :- Stands for no hangup. Starts a process with captures the output in file called as nohup.out. This file is automatically created. 

5.)vmstat :- Shows overall system health dashboard.

6.)du :- shows disk usage

7.) sudo systemctl status <service-name> :- This command shows the current status of the process.

8.) ping :- This command is used to check if the website is reachable or not . Whether the packets are getting tranmitted or not. (Works at network level)

9.)curl :- This command is used to check if the website can respond or not. Checks at applicaion level.

10.)traceroute :- This command show the number of hops it takes to reach the target.

11.) curl ifconfig.me :- Shows the public Ip of your server/device

12.)dig :- Stands for domain information goopher. Used to get information about the domain. Returns the A record of any website.

13.)nslookup :- Stands for nameserver lookup and is useful when we want to get information about the nameserver

14.)cd :- change directory

15.)pwd :- print working directory

16.)mkdir :- Make a new directory

17.)touch <file-name> :- to create a new file 

18.)rmdir <dir-name> :- to remove a directory

19.)rm -rf <dir-name> :- remove the contents of the directory recursively and forcibly

20.)ls :- list the name of the content.
